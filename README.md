# 認定講座構築サイト（確認用プレビュー）

認定講座を作りたい人向けの学習・販売サイトのプレビュー版です。
ネオンちゃん確認用に、デザイン・本文・章構成・スマホ表示をチェックするための静的公開です。

## 構成

- `index.html` — サイト本体（ハッシュルーティングSPA・ビルド不要）
- `data.js` — 教材データ（章構成・本文・商品・料金）
- `images/` — 図版・写真

## 注意（重要）

- このプレビューの権限管理は **localStorage によるモック** です。有料コンテンツを完全に保護するものではありません。
- 本番販売前には、Supabase 等を使ったログイン・権限管理・本文保護の方針を別途検討します。
- 検索避け（`noindex` / `robots.txt`）を設定していますが、URL を知っている人は閲覧できます。共有範囲にご注意ください。

## 閲覧のしかた

- トップから「無料版」はそのまま読めます。
- 完全版（第1〜7章の本文）は、ページ下部フッターの「ログイン」→「完全版購入者としてログイン」（デモ）で閲覧できます。
- または画面下部フッターの「管理画面(モック)」から権限を切り替えられます。
